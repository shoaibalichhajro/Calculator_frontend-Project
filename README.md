A Calculator is a frontend web application that allows users to perform basic arithmetic calculations through a simple and interactive user interface. It takes user input using buttons, displays the entered expression, and instantly calculates the result without requiring any backend or database.

Technologies
HTML5
CSS3
JavaScript (Vanilla JavaScript)

Features
Responsive calculator layout
Number buttons (0–9)
Arithmetic operators (+, −, ×, ÷)
Decimal point support
Clear (C) button
Delete (⌫/Backspace) button
Equal (=) button to calculate results
Real-time display of input and output
Keyboard support (optional)
Error handling for invalid expressions
Mobile-friendly design
